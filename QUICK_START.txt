===============================================================
STEP-BY-STEP SETUP (Takes 5 minutes)
===============================================================

1. INSTALL PACKAGES
   Terminal Command:
   pip install openai python-dotenv sqlalchemy

2. GET OPENAI API KEY
   - Go to: https://platform.openai.com/account/api-keys
   - Click "Create new secret key"
   - Copy the key

3. UPDATE .env FILE
   - Open .env in your editor
   - Find: OPENAI_API_KEY=your-api-key-here
   - Replace with: OPENAI_API_KEY=sk-proj-xxxxx
   - Save the file

4. GENERATE SAMPLE DATA
   Terminal Command:
   python generate_data.py

   This creates:
   - customers.csv (1,000 rows)
   - products.csv (500 rows)
   - orders.csv (5,000 rows)
   - order_items.csv (~10,000 rows)

5. LOAD DATA INTO DATABASE
   Terminal Command:
   python load_data.py

   This creates amazon.db with 4 related tables

6. RUN THE APPLICATION
   Terminal Command:
   python main.py

   Then ask questions like:
   "Show me all customers from Mumbai"
   "What is the total revenue?"
   "List top products"

TROUBLESHOOTING
===============================================================

ERROR: "Module not found: openai"
FIX: pip install openai python-dotenv sqlalchemy

ERROR: "OPENAI_API_KEY not found"
FIX:
  1. Check .env file exists in project root
  2. Verify: OPENAI_API_KEY=sk-proj-xxxxx (not placeholder)
  3. Save file and restart terminal

ERROR: "Database file not found"
FIX: Run these in order:
  python generate_data.py
  python load_data.py

ERROR: "Incorrect API key"
FIX: Get new key from https://platform.openai.com/account/api-keys
